### 해답
```
$ for i in $(cd /usr;echo *); do echo $i ; done
```
### 다른 풀이
```
다른 풀이 1(우에다) $ for f in /usr/* ; do echo ${f##*/} ; done
다른 풀이 2(우에다) $ a=(/usr/*); echo -e ${a[@]/\/usr\//\\n}
```

### 読者別解

by [kariya-mitsuru](https://github.com/kariya-mitsuru)

```
다른 풀이 3 $ (cd /usr; set -- *; IFS=$'\n'; echo "$*")
```
