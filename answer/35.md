### 해답

```
$ cat speech.txt | grep -Po '^.*?[ア-ン]{4}(?=[ア-ン])' | awk '{$2=$1}$1=length($2)'
```

### 다른 풀이

```
[다른 풀이 1] $ cat speech.txt | rb -l "sub(/^((?~[ア-ン]{5})).*/,'\1')" | awk '$0=length($0)" "$0'

[다른 풀이 2] $ cat speech.txt | cure grep -o '^(?~[ア-ン]{5})' | awk '$0=length($0)" "$0'
```

### 독자의 다른 풀이

```
[다른 풀이 3] $ ruby -lpe 'sub(/^((?~[ァ-ヺ]{5})).*/) { "#{$1.length} #{$1}" }' speech.txt
```
