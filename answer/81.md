### 해답
```
$ cat bom.txt | xxd -p | sed "1s/^efbbbf/$(echo -n '[BOM]' | xxd -p)/" | xxd -p -r
```
### 다른 풀이
```
다른 풀이 1（青木）$ nkf --guess bom.txt | awk '$2~/BOM/{printf "[BOM]"}' | cat - <(nkf bom.txt)
다른 풀이 2（山田、上田）$ sed -r 's/\xEF\xBB\xBF/[BOM]/' bom.txt
다른 풀이 3(야마다) $ sed "s/"$'\xEF\xBB\xBF'"/[BOM]/" bom.txt
```
